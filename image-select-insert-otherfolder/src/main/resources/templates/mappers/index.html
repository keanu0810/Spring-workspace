바탕화면에 이미지 저장 폴더 만들기

	사진을 저장하고 불러올 떄 sql 컬럼 자료형으로 verchar2를 사용할 수 있고
	이미지 자체를 저장하는 Blob 을 사용할 수 있음
	db자체에 이미지를 저장하게 되면 이미지가 훼손할 가능성이 있거나 이미지를 글자로 변환할 때
	변환에 문제가 생길 가능성이 높음
	
	db 자체에 이미지를 저장하게 되면 비용이 많이 들기 때문에 많이 사용하지는 않는 방법
	
	이미지를 저장할 때 sql 자료형으로 (varchar2)으로 이미지 경로를 설정해주고 
	설정한 이미지를 가져오는 방식을 진행
	
	이미지가 저장되는 경로
	src/main/resources
			└static
				└image = 홈페이지에서 공개가 가능하고, 공통으로 사용할 수 있는 이미지
	 회사에서 지정한 이미지 저장경로 = 회사가 아닌, 고객이 올리거나 회사에서 비공개로 이미지를 가져오고 
    						   저장하는 경로
